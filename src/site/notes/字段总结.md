==**achievement**==
![Pasted image 20260320102137.png](/img/user/Pasted%20image%2020260320102137.png)


<u>相关逻辑代码位于CashCrazeServer/models/achievement.py</u>

version：achievement.version
version 1：
仅列出部分成就
1_1：用户等级，进度与self.user.level一致
1_2：金币位数
1_14：quest总倍数
2_26：是否加入club
3_46：v_coost激活时间是否大于等于20天
3_50：主题feature收集个数是否大于等于20
<u>详细成就信息可在CashCrazeMath/system_math/a_system_math/achievement/misson_config.py中查看</u>

可简要分类为：
career成就（1_x）：基础信息（vip等级、金币位数等）、登录信息（累计登录、连续登录天数等）、spin信息（spin次数、赢大奖次数等）、社交信息（好友数量、账户绑定情况等）
event成就（2_x）：邮票、quest、card、club等相关信息
hidden成就（3_x）：点赞x次、单日quest spin x次、破产x次等
不同成就可能有不同难度，对应不同奖励

version 2：
新增成就3_57，统计hidden_progress字段中-1的数量，即隐藏任务完成个数
version 4（26-3-10 latest）：
新增付费成就1_26，记录是否使用bucks购买过
调整成就2_10，如果已完成则标记为旧版成就（old_achievement）不做修改，否则将改成就进度值乘以150

record_data：记录最近5次完成的成就信息，格式为[成就编号，成就进度，达成时间]
